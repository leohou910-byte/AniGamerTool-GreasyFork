## 🌟 核心特色

### 1. 🎨 智慧評分徽章與級別變色
* **級別色彩標記**：根據動畫評分自動套用不同色調與框線（神作粉紅、極力推薦金黃、佳作翠綠、普通灰、雷作紅），一眼辨識作品評價。
* **評價防失真機制**：當評分人數少於設定值（預設 800 人）時，會顯示「⚠️」警示並套用黃色警告外觀，避免因少數人評分而產生的分數失真。

### 2. 📊 懸浮五星佔比與分析詳情
* 將滑鼠懸停在評分徽章上，即可顯示懸浮視窗，查看**估算的五星至一星比例分佈**，以及作品的評價分級（例如：神作必看、雷作避難），掌握更詳細的社群反饋。

### 3. 🛡️ 避雷模糊遮罩與完全屏蔽
* **防雷遮罩**：低於設定分數（預設 3.5 分）的作品封面將自動套用半透明模糊與去色效果，避免視覺污染。將滑鼠移到封面上即可暫時解除模糊。
* **完全屏蔽**：低於屏蔽門檻（預設 3.0 分）的作品，可選擇直接從頁面中隱藏。
* **快捷切換**：右下角提供快捷按鈕，可一鍵開啟/關閉遮罩（🛡️）與屏蔽（🚫）狀態。

### 4. 👁️‍🗨️ 已觀看作品淡化與進度徽章
* **視覺淡化**：自動降低已看過或追番中作品的封面透明度，未看過的作品則會加上亮藍色外框突出顯示。
* **進度提示**：在評分徽章下方會顯示您當前的觀看進度（如：「觀看進度：第 5 話」或「尚未觀看」）。
* **雙向同步**：除了自動分析當前網頁結構，亦會在背景安全同步您在動畫瘋的歷史觀看紀錄。

### 5. ⇅ 自訂評分高低排序
* 可一鍵將當前的動畫列表（如首頁新番、分類列表頁）依評分由高到低重新排列。
* 啟用排序後，若遇到尚未載入評分的卡片，腳本會於背景自動預載並顯示頂部進度條，完成後即時重新排序。

### 6. ⚙️ 個人化設定面板與深色模式支援
* **控制面板**：點擊導覽列「⭐ 評分設定」按鈕（若導覽列重構，會自動回退至右下角 ⚙️ 懸浮按鈕）即可開啟設定視窗。
* **細節微調**：可自由調整字體大小、徽章圓角、防雷門檻、載入延遲、快取筆數與有效時間等參數。
* **主題適配**：設定面板外觀會主動跟隨官方的深色/淺色模式，並提供完整的操作導覽。

### 7. ⚡ 智慧快取與高效懶載入
* **智慧懶載入**：採用 `IntersectionObserver` 技術，只有當卡片捲動到畫面上時才發起評分請求，節約網路流量。
* **高效快取保護**：內建 LRU 快取機制，快取有效時間內（預設 24 小時）不重複發送相同請求，並在超出儲存容量上限時自動淘汰最舊快取，避免佔用瀏覽器儲存空間。
